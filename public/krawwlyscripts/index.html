📄 How to Install

1
Install the Tampermonkey browser extension. It's free and works on Chrome, Firefox, Edge, and Safari.
2
Click Copy Script on any script below to copy the full code to your clipboard.
3
Open Tampermonkey and click the + tab (Create a new script). Select all the existing placeholder code and delete it.
4
Paste the copied script, then hit File → Save (or Ctrl+S). Done! Visit the matching Neopets page and it will run automatically.
